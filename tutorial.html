Vantral Tutorial

Komplett veiledning fra installasjon til avansert bruk

1. Installasjon og oppsett

Forutsetninger

Før du starter, må du ha følgende installert:

Viktig: Vantral krever nmap og nuclei for å fungere. Disse må installeres først.

Installere nmap

Last ned fra nmap.org og installer:

nmap --version

Installere nuclei

Med Go installert:

go install -v github.com/projectdiscovery/nuclei/v3/cmd/nuclei@latest

Verifiser installasjon:

nuclei -version

Laste ned Vantral

Last ned vantral.exe fra nedlastingssiden.

2. Din første skanning

Grunnleggende kommando

vantral.exe -d example.com

Dette vil:

  • Laste ned AI-modellen automatisk (kun første gang - ca. 5GB)
  • Starte AI-engine lokalt
  • Kjøre nmap port-skanning
  • Kjøre nuclei sårbarhetsskanning
  • Generere AI-basert sikkerhetsrapport på norsk
Første kjøring: AI-modellen (5GB) lastes ned ved første bruk. Dette kan ta 10-30 minutter.

Uten AI-analyse

vantral.exe -d example.com --no-ai

Rask skanning

vantral.exe -d example.com --fast

3. Forstå resultatet

Resultatene lagres i scan-results-mappen:

scan-results/ └── example.com/ └── 20250916-180604/ ├── security_report.txt # Hovedrapport med AI-analyse ├── nmap_scan.txt # Rå nmap-resultater ├── nuclei_scan.json # Rå nuclei-resultater └── ai_logs/ # AI-engine logger

Hovedrapport

security_report.txt inneholder:

  • Kort sammendrag: Oversikt over sikkerhetsstatus
  • Kritiske funn: Alvorlige sikkerhetsproblemer
  • Anbefalte tiltak: Konkrete forbedringssteg
  • Risikovurdering: LAV/MIDDELS/HØY/KRITISK
Tips: AI-analysen er optimalisert for norske brukere og refererer til relevante norske lover som NIS2-direktivet.

4. Avanserte alternativer

Alle kommandolinjevalg

vantral.exe -d <target> [OPTIONS] ALTERNATIVER: -d, --domain <target> Mål-domene eller IP-adresse --no-ai Hopp over AI-analyse --fast Rask skanning (redusert dekning) -o, --output <dir> Tilpasset utdata-mappe -v, --verbose Detaljert utdata -h, --help Vis denne hjelpen --version Vis versjon

Eksempler på avansert bruk

IP-adresse med verbose output:

vantral.exe -d 192.168.1.100 -v

Tilpasset utdata-mappe:

vantral.exe -d target.com -o ./my-security-reports

Kombinere flagg:

vantral.exe -d example.com --fast --no-ai -v

5. Automatisering

Vantral kan enkelt integreres i scripts og automatiseringsverktøy.

Windows Batch Script

@echo off set TARGET=%1 if "%TARGET%"=="" ( echo Usage: scan.bat <target> exit /b 1 ) echo Starting security scan of %TARGET%... vantral.exe -d %TARGET% --fast if %ERRORLEVEL% neq 0 ( echo Scan failed! exit /b 1 ) echo Scan completed successfully!

PowerShell Script

param( [Parameter(Mandatory=$true)] [string]$Target, [switch]$Fast, [switch]$NoAI ) $args = @("-d", $Target) if ($Fast) { $args += "--fast" } if ($NoAI) { $args += "--no-ai" } Write-Host "Starting scan of $Target..." -ForegroundColor Green & .\vantral.exe @args

6. Feilsøking

Vanlige problemer

"nmap not found"
Installer nmap og sørg for at det er i din PATH. Restart kommandolinjen etter installasjon.
"nuclei not found"
Installer nuclei eller plasser nuclei.exe i samme mappe som vantral.exe.
AI-modellen laster ikke ned
Sjekk internettforbindelsen. AI-modellen er 5GB og krever stabil forbindelse. Bruk --no-ai som alternativ.

Debugging

Bruk -v flagget for detaljert output:

vantral.exe -d example.com -v

Få hjelp

Hvis du fortsatt har problemer:

  • Sjekk GitHub Issues
  • Opprett ny issue med detaljert beskrivelse
  • Inkluder systeminfo og error logs
