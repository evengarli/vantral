Installasjonsveiledning

Installasjon på Windows

1

Installer avhengigheter

Installer nmap og nuclei på systemet ditt:

# Installer nmap fra https://nmap.org/download.html
# Installer nuclei:
go install -v github.com/projectdiscovery/nuclei/v2/cmd/nuclei@latest
2

Last ned Vantral

Last ned vantral.exe og plasser den i en mappe på systemet ditt.

3

Kjør første skanning

Åpne kommandolinje og kjør:

vantral.exe -d example.com

AI-modellen lastes ned automatisk ved første kjøring (ca. 5GB).

Kompilering fra kildekode

1

Klon repository

git clone https://github.com/yourusername/vantral.git
cd vantral
2

Installer Go-avhengigheter

go mod download
3

Kompiler

# Windows
go build -o vantral.exe

# Linux/macOS
go build -o vantral

Systemkrav

  • OS: Windows 10 eller nyere
  • RAM: Minimum 8GB (16GB anbefalt)
  • Lagring: 6GB ledig plass (for AI-modell)
  • Nettverk: Internettilkobling for første oppsett

Påkrevde verktøy

  • nmap: Portskanning og service detection
  • nuclei: Sårbarhetsskanning
  • Go 1.19+: Kun for kompilering fra kildekode

Støttede mål

  • Domener: example.com, subdomain.example.com
  • IP-adresser: 192.168.1.1, 10.0.0.1
  • Subnett: 192.168.1.0/24 (planlagt)
  • Protokoller: HTTP, HTTPS, alle TCP-porter
